# How to Install the Job Agent Chrome Extension

## Steps (takes 2 minutes)

1. Open Chrome → go to `chrome://extensions`
2. Turn ON **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select this folder: `chrome-extension/`
5. The extension appears in your toolbar 🤖

## First-time setup

1. Click the 🤖 icon in Chrome toolbar
2. Enter your app URL: `http://localhost:3000`
3. Click **Connect**

## What it does

- **On any job page** (LinkedIn, Indeed, Greenhouse, Lever, Glassdoor):
  - Shows a floating "Job Agent — Check Match" badge
  - Click the extension icon → see your AI match score instantly
  - Save the job to your Job Agent database
  - One-click open in dashboard

## Icons

You need 3 icon files in the `icons/` folder:
- `icon16.png` (16×16)
- `icon48.png` (48×48)
- `icon128.png` (128×128)

You can use any robot/briefcase emoji as PNG, or generate them at:
https://favicon.io/emoji-favicons/robot/
